# 📐 Calculadora de Áreas Geométricas

Este é um projeto prático desenvolvido como parte dos desafios da plataforma **[Codédex](https://www.codedex.io/)**. 

O objetivo do exercício é aplicar conceitos fundamentais de lógica de programação, estruturas de repetição (`while`) e condicionais (`if/elif/else`) para construir uma ferramenta interativa via linha de comando (CLI).

---

## 🚀 Funcionalidades

A calculadora interativa permite ao usuário escolher uma forma geométrica e calcular sua respectiva área com base nas entradas fornecidas:
- [x] **Triângulo** (Base e Altura)
- [x] **Retângulo** (Comprimento e Largura)
- [x] **Quadrado** (Lado)
- [x] **Círculo** (Raio)
- [x] **Tratamento de erros** para opções inválidas no menu.
